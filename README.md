# Frontend

월급쟁이부자들 F/E 과제용 강의 등록/신청 프론트엔드입니다.  
회원가입, 강의 등록(강사), 강의 신청(수강생/강사 모두) 기능을 제공합니다.  
요구사항은 과제 문서인 `FRONT_README.pdf`를 기준으로 구현했습니다.

## 주요 기능

### 1) 회원 가입

- 이름, 이메일, 휴대폰 번호, 비밀번호, 회원 유형(수강생/강사) 입력
- 가입 완료 후 강의 목록으로 이동

### 2) 강의 등록 (강사 전용)

- 강의명, 최대 수강 인원, 가격 입력
- 강의 등록 완료 후 강의 목록으로 이동
- 수강생은 접근 불가

### 3) 강의 신청

- 무한 스크롤
- 정렬 옵션: 최근 등록순 / 신청자 많은 순 / 신청률 높은 순
- 강의 다중 선택 후 한 번에 신청 가능
- 강사 회원도 수강 신청 가능

## 기술 스택

- React (Next.js App Router)
- TypeScript
- React Query

## 실행 방법

### 1. 의존성 설치

```bash
pnpm install
# 또는
npm install
```

### 2. 개발 서버 실행

```bash
pnpm dev
# 또는
npm run dev
```

## API 서버

- FSD 2.1 기반 프로젝트 구조를 따릅니다.

```

  entities/

  views/
  shared/
  widgets/
```

## 요구사항 대응 정리

- 회원 가입 / 강의 등록 / 강의 신청 UI 및 동작 구현
- 정렬 옵션 제공
- 무한 스크롤 적용
- 모바일 웹 기준 레이아웃 고려
- SSR 및 React Query 프리패치 사용

## 추가 구현 사항

- 수강 신청 결과 피드백(성공/실패/부분 성공 메시지) 제공
- 마감된 강의 자동 선택 해제 및 선택 불가 처리
- 로그인 유도 후 이전 선택 복원(수강 신청 미완료 상태 유지)
- 체크아웃 바의 상세 선택 목록 토글(더보기/접기)
- 접근 권한 가드(강사 전용 강의 등록 화면)
